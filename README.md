# SCP in Minecraft

## Demo
Go in the release tab and download both the resource pack and the datapack. Extract it in the correct folders.

## What is SCP

link: **https://scp-wiki.wikidot.com**

use version 1.21.11/.10/.09

## command:
if you wanna test the stuff in creative type the command: 
```
function scp:demo
```
it will give you all the new items from the datapack and spawn all the new mobs that are added by this datapack.

- if it is a mob just check in the functions if the end is ... _summon
If you want to:

- use 117: sneak+click to change the tool you use

- use 122: place it anywhere it will get rid of the monsters
